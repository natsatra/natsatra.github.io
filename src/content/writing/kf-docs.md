## Overview

A selected set of admin guides I wrote for Kissflow Apps, Kissflow's low-code application development platform. These cover core building blocks — variables, cross-app data lookups, and custom form field components — for admins and citizen developers building and extending apps on the platform.

* A comprehensive guide to using variables in Kissflow Apps' development environment: <a href="https://community.kissflow.com/t/83yzj4y/using-variables-in-application-development" target="_blank" rel="noopener noreferrer">Variables in Kissflow Apps</a>

* A detailed guide on how to link two applications and securely cross-reference data through lookups: <a href="https://community.kissflow.com/t/35yzjvd/application-to-application-lookup-through-linking" target="_blank" rel="noopener noreferrer">App-to-app lookup through linking</a>

* A detailed guide on building custom form fields using React JS and Kissflow's native SDKs: <a href="https://community.kissflow.com/t/35yfjp0/working-with-custom-form-field-components" target="_blank" rel="noopener noreferrer">Building custom form fields</a>
